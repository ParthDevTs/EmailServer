Node Mailer Server

This is a server Made using Node Js and Express to Send Emails from a contact form in your webpage and store email id in Database using MongoDB

There are two Apis that this server exposes:

  • Send Email API

    This Api is responsible for send mails after receiving a JSON object with email details and stores Email Address and Username in the server.

  • Get Statistics API

    Returns a JSON object with Total Hits on the API and Number of Unique Email Id present

Note: We dont store any senstive data and all storage data is merely for statistical purposes
